# Blockchain-Based Online Voting System

A secure and decentralized online voting platform developed as a Final Year B.Tech Project. The system leverages Ethereum blockchain technology and smart contracts to provide transparent, immutable, and tamper-proof elections.

## Features
- Secure voter authentication using MetaMask
- Blockchain-based vote recording
- One Person, One Vote mechanism
- Candidate management by administrator
- Election start/stop control
- Real-time vote counting
- Transparent result declaration
- Blockchain audit trail
- SHA-256 based voter identity hashing

## Technology Stack
- Frontend: React.js, Vite, Tailwind CSS
- Blockchain: Ethereum Sepolia Testnet
- Smart Contract: Solidity
- Wallet: MetaMask
- Library: Ethers.js
- Development Tools: VS Code, Node.js
